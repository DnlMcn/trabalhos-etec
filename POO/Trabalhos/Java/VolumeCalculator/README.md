Trabalho escolar efetuado sob orientação da professora Nívia Maria Domingues.

Esse programa calcula o volume de um retângulo (piscina) em metros cúbicos, dados o comprimento, largura e profundidade da mesma.
Então, calcula um valor para a água para preenchê-la, multiplicando o volume pela constante nos dada pela professora (R$300,00). 
Script, comentários e interface do usuário 100% em inglês.
