# trabalhos-etec
Repositório pessoal para organizar e efetuar trabalhos escolares da ETEC.

Organizado em 4 pastas, categorizando trabalhos em progresso, e subpastas para cada matéria. Cada trabalho deve ser sua própria pasta, com um .txt chamado "Instrucoes," contendo uma descrição do que deve ser feito. Mais duas pastas, uma para materiais de referencia, e outro para arquivos criados durante a efetuação do trabalho (desnecessário para trabalhos menores).
